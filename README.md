# AdministradorTrafico
## Uso
Programa creado para realizar un Traffic Shadowing donde replicara los datos 
insertados en la primera maquina virtual 
### Formato de ejecución del programa
``` 
sudo java AdministradorTrafico <host-principal> <puerto-principal> <ip-replica> <puerto-replica> <puerto-local>"); 
```

### Datos adicionales 
Java 21
